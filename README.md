##  Quick Start

Follow these steps to set up project on your computer

**Clone the Repository**

```bash
git clone https://github.com/PaulaIbagere/Brainwave-website.git
cd Brainwave-website
```


**Install Packages**
```bash
npm install
```

**Run the Project**
```bash
npm run dev
```
